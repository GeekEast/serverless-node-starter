## Usage
### Install
```sh
yarn
```

### Import
```yml
plugins:
  - serverless-plugin-typescript
  - serverless-offline-schedule
```

### Execute
```sh
serverless offline
yarn dev
```

### Deploy
```sh
sls deploy
```

### Logging
```sh
sls log -f hello -s dev -t
```

### Remove
```sh
sls remove
```
